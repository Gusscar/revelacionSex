# PROJECT

Create a viral, modern, mobile-first PWA for Baby Gender Reveal events.

The application must feel emotionally exciting, highly animated, social, memeable, and optimized for TikTok/Instagram sharing.

Main objective:
Create a product people WANT to share.

Tech stack required:
- Next.js latest (App Router)
- TypeScript
- TailwindCSS
- Supabase
- Framer Motion
- Zustand
- React Hook Form
- Zod
- PWA support
- Mobile-first responsive design

---

# PRODUCT IDEA

Users create a Baby Reveal Event.

Friends and family join through:
- invite link
- QR code

Participants can:
- vote Team Boy or Team Girl
- react live
- send emojis
- upload reaction videos/photos
- participate in mini-games
- view countdown
- unlock fake reveals
- watch final reveal animation

The app should generate viral/shareable content automatically.
